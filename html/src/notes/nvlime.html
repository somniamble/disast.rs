Starting up a lisp image with nvlime running

This assumes you already have nvlime cloned to e.g. ~/common-lisp

To get a server running on port 7002 (by default) 

sbcl --load $HOME/common-lisp/nvlime/lisp/start-nvlime.lisp \
# ... then whatever else

If there's already something on that port, you'll get a USOCKET:ADDRESS-IN-USE signal, and a restart [0] to choose a different port. Pretty nice?
